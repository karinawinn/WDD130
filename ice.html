In Class Exercises

Prophet Card

In this exercise, I created html and css code that would format and style the page to match the provided wireframe. Throuhgout this process, I learned how to intentionally structure html to make styling easier by creating div containers with specific ids. I also learned how to style a page using flexbox tools.

Apostle Spotlight

The purpose of this exercise, much like the previous one, was to create html and css code that would create a page that follows the provided wireframe. From this exercise, I learned how to style a page using grids, create image borders of different shapes, and layer them using z-index.

Favorite Devotional Table

This exercise is meant as a way to practice making tables and forms. And that is exactly what I was able to take away from this exercise. I learned how to create and style forms and tables using html and css.

Holiday Page - Fathers Day

During this exercise, I used only css to alter the webpage to create a Father's Day card. This showed me just how much the css impacts the entire look of the webpage. I also learned how to style an unordered list horizontally and without bullet points in css.

Country Flags with a Grid

This exercise was a way to practice using css grids by creating the flags of different countries. This taught me how to make elements span multiple rows and columns in addition to teaching me how to utilize gaps in css grid structure.

CSS Positioning Activity

In this exercise, I used css grid structure to recreate the layout from six different images. By doing this, I learned how to plan and create grid structures to achieve the desired results. I also learned how to use flexbox to wrap text around an element.

Image Gallery Issues

The purpose of this exercise was to optimize images on a webpage so that it would load faster. Throught this exercise, I learned how to use the Image Size function in photo editors like Photopeia and Adobe Express to reduce the image size. I also learned that image files like JPG and webP are lower quality than PNG images but are still usable on web pages.

Signup Form Design

In this final exercise, I adjusted the provided html and css to style a form so it would match the provided image. Doing this taught me how to utilize containers to create several grids for a more complicated structure. I also learned how to use relative positioning to move a button contained in the form to a location outside the form.
